# LRU Cache Visualizer

An interactive **LRU (Least Recently Used) Cache Visualizer** built using **React** to understand how caching works internally with real-time animations and history tracking.

##  Features

- Visual LRU Cache behavior
- PUT / GET operations
- Cache hit, miss & eviction statistics
- Real-time cache usage indicator
- Timeline slider to review cache history
- Capacity lock after first operation
- Fully responsive (mobile + desktop)
- Clean, modern UI with smooth animations

##  Tech Stack

- Frontend: React (Hooks, Context API)
- Styling: CSS (Custom, Responsive)
- Logic: Custom LRU Cache implementation (Map + Doubly Linked List concept)

## How to Run Locally
git clone https://github.com/nishmitha1112/lru-cache-visualizer.git
cd lru-cache-visualizer
npm install
npm start
